FitSphere – Full Stack Fitness Web Application
Project Overview
FitSphere is a full-stack web-based fitness management application designed to help users track workouts, manage health-related data, and monitor fitness progress. The project integrates a Python-based backend with a responsive frontend to provide a seamless user experience.
The system demonstrates backend development, frontend integration, user session handling, and structured web application architecture.

Features
•	User Registration and Login Authentication
•	Secure Session Management
•	Workout Tracking and Fitness Data Management
•	Dynamic Template Rendering
•	Form Handling and Input Validation
•	Modular Backend Route Structure
•	Responsive UI using HTML, CSS, and React
